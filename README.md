Deskripsi Program:
1. Parkir Mall "Prienal"
   - Sistem pengelolaan parkir kendaraan, petugas parkir, dan perhitungan biaya parkir.

2. Game Tebak Kata
   - Permainan menebak kata dengan sistem nyawa, tingkat kesulitan, dan pemilihan kata secara acak.

3. Sistem Manajemen Inventaris Toko
   - Program pengelolaan data barang, stok, dan nilai inventaris toko.

4. Minesweeper++
   - Permainan Minesweeper berbasis CLI dengan papan dinamis, sistem penandaan bom, dan penghitung waktu.

5. Game Tebak Negara
   - Permainan edukatif untuk menebak nama negara berdasarkan petunjuk yang diberikan.
